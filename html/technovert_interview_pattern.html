Programming Test

  • Reverse each word in a string but the words should be in same order i.e., hi how are you -> ih woh era uoy
  • Return the Most repeated number in a string but it should be not case-sensitive i.e., ‘e’ is equal to ‘E’ Ex: We are Codehub Output: “E”

System Test

  • Implement File system/File Directory
    It should give following options first:
    1.add folder
    2.add file
    3.delete folder (it’s child folders should be also deleted)
    4.delete file
    5.show the folder structure
    Ex:
    Root
    Folder-1
     file
     file
     Folder-2
     File
     note: Root folder should be the parent folder
